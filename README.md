# 📄 A-Resume

A clean, responsive, and web-based resume built using **HTML & CSS**. This project showcases personal details, skills, professional experience, education, and projects in a modern, easy-to-read layout — perfect for online portfolio integration.

---

## ✨ Overview

This is a simple portfolio resume website aimed at providing a professional online presence. It is built with **pure HTML and CSS**, making it lightweight and easy to customize. The design is fully responsive, ensuring optimal viewing across desktop, tablet, and mobile screens.

---

## 🧠 Features

✔ Simple yet elegant professional resume UI
✔ Fully **responsive layout** (mobile & desktop friendly)
✔ Structured sections for personal info, skills, experience, education, and projects
✔ Clean and readable styling with pure CSS
✔ Easy to customize and extend

## 🚀 Getting Started

Follow these steps to run or customize this project locally:

### 1. Clone the Repository

```bash
git clone https://github.com/singhaditya55/a-resume.git
cd a-resume
```

### 2. Open in Browser

Just open the main file:

```bash
open index.html
```

or open `index.html` in your favorite browser.
